## 📋 Deploy Day (When Site Goes Live)

### Immediate (5 min)

- [ ] **Submit sitemap to Google Search Console**
  - Go to: search.google.com/search-console
  - Select property: `vapewissen.de`
  - Sitemaps → Add new sitemap
  - Enter: `https://vapewissen.de/sitemap.xml`
  - Submit

- [ ] **Request indexing** for key pages:
  - In Search Console → URL Inspection
  - Test each URL:
    - `https://vapewissen.de/`
    - `https://vapewissen.de/artikel/temperatur-guide/`
    - `https://vapewissen.de/artikel/cannabinoide-guide/`
  - Click "Request Indexing"

- [ ] **Verify robots.txt** accessible:
  - Visit: `https://vapewissen.de/robots.txt`
  - Should allow Googlebot

- [ ] **Add structured data** (optional, but helpful):
  ```html
  <!-- In article frontmatter -->
  <script type="application/ld+json">
  {
    "@context": "https://schema.org",
    "@type": "Article",
    "headline": "{{ .Title }}",
    "description": "{{ .Description }}",
    "datePublished": "{{ .Date }}"
  }
  </script>
  ```

- [ ] **Add FAQ sections** to articles:
  - FAQ schema markup helps rich snippets
  - Already have FAQs in content, just need markup
